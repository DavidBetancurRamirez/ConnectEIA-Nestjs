# Plantilla Nest.js
Plantilla usada para proyectos de nest.js


## Ejecutar
**Levantar BD con docker:**
```
  docker compose up -d
```

**Descargar dependencias:**
```
  npm i
```

**Ejecutar:**
```
  npm run dev
```
